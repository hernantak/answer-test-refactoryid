# Skill Test From Refactory.id -- Q1

This is a payment receipt program for a restaurant.
There are five pieces of input that must be filled in.
There are :
1. Resto name.
2. Date of print.
3. Cashier name.
4. The item.
5. The price.

How to build :
1. Create folder build using ```system mkdir build```
3. ```cd build/```
4. ```system cmake ..```
5. ```system make```

How to run :
1. ```system cd build/```
2. ```./run-q-1```
